简单版的购物商城，用到了大量框架；其中购物车的逻辑比较复杂，主页的不同adapter值得借鉴；
# 详细笔记

2. 替换 VIewpager 的Banner 动画；

5. FloatBbutton 的使用，
6. 以及访问本地 TomCat 服务器获取数据(事实上，应用中用到的数据全部来自于本地 TomCat 服务器)；
